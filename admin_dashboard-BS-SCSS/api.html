Conecta Tu Tienda

Utiliza nuestras herramientas para integrar tu tienda online (WooCommerce, Prestashop, Shopify o una solución personalizada) y gestionar tus datos a través de nuestra API.

Conectar WooCommerce

Una vez generada la API key, siga estos pasos para conectar su tienda WooCommerce:

En WooCommerce, diríjase a WooCommerce > Ajustes > Avanzado > REST API y haga clic en Añadir clave. Ingrese los permisos necesarios y copie la clave generada.

Pegue la API key en el panel de integración de su CRM. La API key se enviará en el encabezado Authorization: Bearer [API_KEY] en cada solicitud.

Para más información, consulte la documentación oficial de WooCommerce.

Conectar Prestashop

Siga los siguientes pasos para conectar su tienda Prestashop:

Vaya a Parámetros Avanzados > Webservice y active el acceso a la API. Genere una nueva clave de API.

Configure los permisos adecuados para la API key y cópiela en el panel de integración. La API key se enviará en el encabezado Authorization: Bearer [API_KEY].

Para más información, consulte la documentación de Prestashop.

Conectar Shopify

Para conectar su tienda Shopify, siga estos pasos:

En su panel de Shopify, vaya a Apps > Administrar apps privadas y cree una nueva aplicación privada para generar la API key y contraseña.

Inserte la API key y contraseña en el panel de integración de su CRM. En cada solicitud, envíe estas credenciales en el encabezado.

Para más detalles, consulte la documentación oficial de Shopify.

Conectar Tienda Personalizada

Si utiliza una solución de comercio electrónico personalizada, puede integrar la API siguiendo estos pasos:

Asegúrese de que su tienda tenga un endpoint REST o GraphQL configurado para recibir solicitudes. Configure la autenticación con una API key.

Ingrese la API key en su sistema CRM y realice pruebas de conexión. Verifique que cada solicitud incluya el encabezado Authorization: Bearer [API_KEY].

Ajuste estos pasos según la arquitectura de su tienda personalizada.
